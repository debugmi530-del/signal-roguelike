# SIGNAL

Бесконечный рогалик про эхо радиоволны, застрявшее в вечном эфире.
Полностью управляется геймпадом: левый стик — движение, правый стик/RT/RB — стрельба,
Start — пауза, A — подтвердить, B — назад.

## Структура проекта

```
game/
├── project.godot          # конфигурация проекта, автозагрузки
├── autoload/               # синглтоны: StatManager, CardDatabase, StationDatabase, EnemyDatabase, GameManager
├── scripts/                # логика игрока/врагов/боссов/снарядов/меню
│   └── ui/                 # HUD, экран выбора карты, пауза, экран смерти, настройки
├── scenes/                 # .tscn (только корневой узел + скрипт, остальное строится кодом)
├── systems/                # генератор комнат станции
├── resources/               # контент как данные, а не код
│   ├── cards/               # ~270 карт (14 базовых модулей + 10 групп карт-жертв)
│   ├── stations/            # 8 тематических станций
│   ├── enemies/             # типы врагов
│   └── bosses/              # боссы
├── tools/generate_content.js  # перегенерирует весь пул карт/станций/врагов/боссов
└── installer/setup.iss     # сборка Windows-инсталлятора через Inno Setup
```

## Запуск в редакторе

Открыть папку `game/` в Godot 4.3+ и нажать Play. Главная сцена — `scenes/MainMenu.tscn`.

## Сборка Windows-инсталлятора

Собирается автоматически через `.github/workflows/build.yml` при пуше тега `v*`
или вручную (workflow_dispatch): экспорт Godot → Inno Setup → GitHub Release.

## Дизайн (кратко)

- Вид сверху, только дальний бой (снаряды), как Isaac Rebirth.
- Забег полностью бесконечный, без постоянной прокачки между забегами —
  единственное, что переносится между попытками, это пул карт, открывающийся
  по мере продвижения (показывается на экране смерти).
- Карты складываются аддитивно и всегда показывают итоговое значение статы
  прозрачно (превью на карте).
- Комнаты станции: 1 комната босса, 0-2 магазина, 0-4 сокровищницы,
  5-25 комнат с врагами; зачищать не обязательно, чтобы пройти дальше.
- Боссы — каждые 5 станций.
